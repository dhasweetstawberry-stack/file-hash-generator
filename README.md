# 🔐 File Hash Generator (SHA256)

A lightweight Python-based CLI tool to generate SHA256 hash values for files.  
Useful for file integrity verification, cybersecurity projects, and digital forensics.

---

## 🚀 Features

- Generates SHA256 hash
- Efficient chunk-based file reading (handles large files)
- Error handling included
- Simple command-line interface
- No external dependencies required

🔐 What is SHA256?
SHA256 (Secure Hash Algorithm 256-bit) is part of the SHA-2 family developed by the National Security Agency (NSA).
It produces a 64-character hexadecimal hash value that uniquely represents file data.
